Relative Units

%

% of parent element

em

em is relative to the font size of current element

rem

rem is relative to the font size of the root element

vh/vm

vh is relative to viewport's height

vw is relative to viewport's width

position properties

static

  • Element will be at its normal position
  • Scrolling will work normally for it

relative

relative

  • Element will be relative to its normal position
  • Scrolling will work normally for it
  • Here, relative sub heading is position:relative, top:2px and left:2px.

absolute

absolute

  • Element will be relative to the top left corner of the webpage
  • Scrolling will work normally for it
  • Here, absolute sub heading is position:absolute, top:791px and left:10px.

fixed

fixed

  • Element will be fixed in the webpage
  • Scrolling won't affect it
  • Here, fixed sub heading is position:fixed, bottom:10px and right:10px.
