My color is defined by Universal Selector

Types of Selectors:

  1. * is Universal Selector
  2. Type Selector e.g.=> header {}
  3. ID Selector e.g.=> #<id value>
  4. Class Selector e.g. => .<class value>
  5. Group Selector e.g. => h1, h2, h3 {}
