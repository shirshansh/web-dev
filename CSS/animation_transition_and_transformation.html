Animation, Transition and Transformation

Pseudo Classes

Pseudo-classes are used to style elements based on their state. They are not actual HTML elements but keywords that are used to select elements. For e.g:- :hover, :active

Transition

Transition is a CSS property that lets you specify how a property changes when the property is being animated. It's used to smoothly animate changes between the values of CSS properties.

  • transition-property: The CSS property you want to animate.
  • transition-duration: The duration of the animation.
  • transition-timing-function: The timing function for the animation.
  • transition-delay: The delay before the animation starts.

Transform

Transform is a CSS property that lets you manipulate an element's position, size, shape, and color. It's used to create 3D transformations and animations.

Animation

Animation is a technique used to create dynamic effects on a web page. It involves creating a set of CSS rules that define the animation, and then applying that animation to HTML elements. 


Syntax: @keyframes <function> {
            from { /* starting state */ }
            to { /* ending state */ }
        }
